Speed Typer
===
A terminal typing accercy test for fun and profit.

Usage: spd 

Keys:
  CTRL-C / Q: Quite the game
  Escape    : Go back a screen
  M         : Game settings
  L         : Load a typing file (todo)
  S         : Start game    
  C         : Conways game of life (why not)

A basic typing game that counts accarcy.
